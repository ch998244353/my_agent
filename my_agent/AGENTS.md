# my-agent Agent Notes

This repo is a Python teaching implementation of an Agents SDK-style runtime: agents, model turns, tools, guardrails, workspace context, sessions, tracing, and verification.

Start with [docs/llm/ARCHITECTURE_INDEX.md](docs/llm/ARCHITECTURE_INDEX.md) for LLM-facing architecture context.

When changing runtime behavior, public symbols, state models, tool contracts, or tests, update the relevant `docs/llm/*` file and append `docs/llm/MAINTENANCE_LOG.md`.

Code beats docs. If docs and code disagree, trust code and fix the docs.
